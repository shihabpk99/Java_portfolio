# Student Management System (Java)

A simple **console-based Student Management System** written in Java.
This project demonstrates core Java concepts such as **OOP**, **file handling**, and **menu-driven program flow**.

---

## Features

* Add new students
* View all students
* Search student by ID
* Remove student by ID
* Persistent storage using a text file (`students.txt`)

---

## Technologies Used

* Java (JDK 8 compatible)
* File I/O
* Object-Oriented Programming
* ArrayList
* Console input (`Scanner`)

---

## Project Structure

```
java-portfolio/
└── Student_Management_System/
    ├── Main.java
    ├── Student.java
    ├── StudentManager.java
    └── students.txt
```

---

##  How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/java-portfolio.git
```

2. Navigate to the project folder:

```bash
cd Student_Management_System
```

3. Compile the program:

```bash
javac --release 8 *.java
```

4. Run the program:

```bash
java Main
```
